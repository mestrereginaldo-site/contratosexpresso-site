# Contratos Expresso

Site oficial da Contratos Expresso - Modelos de contratos jurídicos prontos com IA.

## 🚀 Sobre
Plataforma de contratos jurídicos pré-formatados com IA integrada para personalização instantânea.

## 💻 Tecnologias
- Next.js 14
- Tailwind CSS
- Vercel (Deploy)

## 📦 Instalação
\`\`\`bash
npm install
npm run dev
\`\`\`
